# Firebase Playground Nodejs

## Setup
1) ``` npm install ```

2) Create firebase account [Firebase Setup](https://firebase.google.com/docs/admin/setup)

3) ``` npm install firebase-admin ``` 

4) ```.env``` for firebase ```services.json``` must in one line



## Run
1) Run
```
npm start
```

2) Dev Nodemon
```
npm run dev
```
